Code

Dependencies

restify = require 'restify'

Create the server here. Later I will add code to load config.

server = restify.createServer
  name: 'RewindBBS'

Set the charset of response to ‘utf-8’

server.use (req, res, next) ->
  res.charSet 'utf-8'
  next()
